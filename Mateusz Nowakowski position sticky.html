Static

Element statyczny nie jest pozycjonowany w jakiś szczególny sposób

Position static

Relative

Ustawienie top, right, bottom lub left elementu pozycjonowanego "relative"" spowoduje, że zostanie on odsunięty od jego normalnego położenia. Inna zawartość nie zostanie dopasowana do luki pozostawionej przez element.

Position relative

Fixed

Element z pozycją: fixed; jest pozycjonowany względem widocznego obszaru, co oznacza, że zawsze pozostaje w tym samym miejscu, nawet jeśli strona jest przewijana. Właściwości top, right, bottom i left służą do pozycjonowania elementu. Stały element nie pozostawia luki na stronie, w której normalnie by się znajdował.

Position fixed

Absolute

Element z pozycją: absolutny; jest pozycjonowany względem najbliższego pozycjonowanego przodka jeśli element pozycjonowany bezwzględnie nie ma pozycjonowanych przodków, używa treści dokumentu i porusza się wraz z przewijaniem strony.

Div relatywny

Sticky

Element z pozycją: sticky; jest pozycjonowany na podstawie pozycji scrollowania przez użytkownika.

Try to scroll inside this frame to understand how sticky positioning works.
